## 特殊说明

### 消费者暂停恢复

* 消费者直接暂停消费：生产者生成数据、消费者接收数据
* 媒体服务暂停消费者数据转发（默认）：生产者生成数据、消费者不会接收数据

### 生产者暂停恢复

* 消费者直接暂停接收：生产者生成数据、消费者接收数据
* 媒体服务暂停生产者数据转发：生产者生成数据、消费者不会接收数据
* 媒体服务暂停生产者数据转发同时暂停生产者生成（默认）：生产者不会生成数据、消费者不会接收数据
