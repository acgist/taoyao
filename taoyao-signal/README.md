# 信令

## 格式

```
{
	"header": {
		"v": "版本",
		"id": 请求标识,
		"sn": "设备标识"
		"pid": 信令标识,
	},
	"code": "响应编码",
	"message": "响应描述",
	"body": {
		// 信令参数
	}
}
```
